# AI-Powered Resume Analyzer

## Overview
A hybrid Resume Analyzer built using Python that evaluates resume-job alignment using:

- Skill-based keyword matching
- TF-IDF vectorization
- Cosine similarity
- Weighted scoring system

## Features
- Extracts predefined technical skills
- Computes skill match percentage
- Calculates semantic similarity
- Generates structured analysis report

## Technologies
- Python
- scikit-learn
- NLP (TF-IDF)
- Set operations

## How to Run

1. Clone the repository
2. Create virtual environment
3. Install dependencies:

pip install -r requirements.txt

4. Run:

python resume_analyzer.py
